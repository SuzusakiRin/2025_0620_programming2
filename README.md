**演習７－１**
main関数において、int型の１２個分の
領域を作り、そのアドレスをポインタ変数
pに代入する。確保した領域に、適当な整
数値を代入する。最後に、１２個の値を３
行４列の行列と見立てて標準出力せよ。ま
た、以下の条件に従うこと。
• 整数値の代入は、標準入力とする。
• 標準出力では2重のfor文を使う。
• 標準出力では配列表記を使う。
⇒ p[i]のような表記を使う。

**演習７－２**
main関数において、int型の１２個分の
領域を作り、そのアドレスをポインタ変数
pに代入する。確保した領域に、適当な整
数値を代入する。最後に、１２個の値を３
行４列の行列と見立てて標準出力せよ。ま
た、以下の条件に従うこと。
• 整数値の代入は、標準入力とする。
• 標準出力では1重のfor文のみで行う。
⇒改行判定は余算「％」を用いる。
• 標準出力では配列表記を使わない。
⇒ *(p+i)のような表記とする。
